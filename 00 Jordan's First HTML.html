Sample HTML

Introduction to Basic and Usual HTML Codes

These are a simple HTML programming codes that I have learned.


Table 1: Sample Codes
Code Name Category Meaning
1 h1 to h6 Header Element It is used to make a context a header. The higher the sequence number, the smaller the font size. Should have a closing tag.
2 br Page Break Element It is the same as the "Enter" function in typing --- going to the next line. Should have a closing tag.
3 hr Horizontal Line Element Inserting a straight horizontal line. Self-closing tag.
4 size="n" Context font size Attribute Changing the font size of the context. n means the size number (Ex: size"3"). This is a self-tag; does not need a closing tag.
5 center Center Element It is used the center the context. Should have a closing tag.
6 table Table Element Inserting a table in the page. Should have a closing tag.
7 tr Table Row Element Selecting the row for the context. Starts with the topmost row. Should have a closing tag.
8 th Table Header Element Used to create a header. Can be either a table row or a column. Usually it is being followed by an attribute scope="row or col".
9 td Table Context or Data Element Use to input a context in a particular cell in the table. The entry is per row, starting from the leftmost to the rightmost. Should have a closing tag.
10 caption Caption Element Used to display the table's caption. Usually can be found at the bottom of the table. Should have a closing tag.
11 scope="col or row" Column or Row header Attribute Used to identify if the header is for a row or for a column. Self-closing tag.
12 !--context-- Comment Element Use to display a comment on the program but will not be shown in HTML
13 noshade No shade Attribute Used to shade the line such as the horizontal line
14 class Class Division Grouping Attribute Use to declare or call a certain divisions under the same class.
15 div Division Element Putting a certain group of codes into one container. It is like creating to declare a group or block.
16 id Identification of a division Attribute Use to name a single division or group. It is use to call-out a certain division for editing.
17 ul Unordered List Element Used to create a list of bullets. Needs to have a closing tag and the bullets is being coded using "li" with closing tag also.
18 ol Ordered List Element It is the opposite of UL wherein the list is being displayed in numbers instead of bullets. Should also have a closing tag and use "li" as well. May style the numbering, may use roman numeral simply putting the attribute of type="I"
19 p Paragraph Element Use to do paragraph the style of the context.
20 em Itilicize Element Use to itlicize the selected context. Should have a closing tag.
21 strong Bold Element Used to make the selected context format to bold. Should have a closing tag.
22 img Image Element It is used to insert an image in HTML. Should always have an attribute src (source) to link the image. Use the attribute alt to display a label in case the image does not load; so the viewer can immediately identify what the unloaded photo is.
23 link Link Element Used to link a css file in the html file. Format has an attributes link rel="stylesheet" href="type the link here"
24 a Linking a contect or an image Element Use to create a hyperlink-like on the selected context or image. Has a closing tag. Format is (a href="link here")
25 target Target action Attribute This is used to set a target to be done once the element code is triggered. Usually used when the target context will be opened in a new tab, code is target="_blank".
26 href Reference Link Atribute Used to link the context
27 src Source Attribute Used to identify or locate the source of an image.
28 style Putting style in HTML Element Used for putting a simple css codes in HTML
29 input Interactive Controls Element Add an interactive controls like, but not limited to, date, button, volume scroll, choose file, etc. These are the attributes. Has no closing tag. Should have an opening and closing tag of "form"
